---
title: 虚拟机迁移
---

- [Proxmox Backup Server](https://pbs.proxmox.com/wiki/Main_Page)

在源虚拟机控制页面中，对虚拟机进行备份。

备份目录一般在：`/var/lib/vz/dump`

备份完成后会生成一个带有 VM ID 的压缩文件，将文件拷贝到新主机，在新 PVE 主机中执行恢复。

```sh
# 恢复至空闲的 VM ID
# 备份文件需删除前缀，以 vzdump 开头

# 命令行恢复
qmrestore vzdump-qemu-2025_11_19-21_53_17.vma.zst 101

# 也可以将备份文件放入 /var/lib/vz/dump，并在 Web 页面操作
```
